■内容
１．L字やT字などのテトリミノを作ろう！


２．テトリミノをランダムで出現させよう！
・SpawnMono.cs を Hierarchy へ
・Inspectorにて
　┗Z座標をBackGroundよりも手前の 9 にする(0, 10, 9)
　┗１．で作ったMinosをすべて追加
Code:
    int randomIndex = Random.Range(0, Minos.Length);

３．テトリミノを自動で落下させよう！
・MinosのすべてのMinoに、Monos.csを付与（AddComponent）
・GameManager.cs を Hierarchy へ
・Inspectorにて
　┗ Auto Fall Data List(0, 1)
　
// 自動落下コード
        if (gameManager.GetAutoFallTime() > 0 && Time.time - previousTime > gameManager.GetAutoFallTime())
        {
            addVector = new Vector3(0, -1, 0);
            moveOn = true;
            previousTime = Time.time;
        }


４．テトリミノをキーで移動・回転させよう！
・Inspectorにて
　┗Mino Config List で +


５．サウンド（SE）を鳴らそう！


６．スコアを表示しよう！


７．スコアによりテトリミノの落下速度を変化させよう！


８．エフェクトを付けてみよう！
・EffectフォルダにあるBurstParticleSystemをHierarchyへ
・エフェクトのPlay再生
・Global Volume追加
　・InspectorのNewボタンでProfile追加、Additiveボタン
　・Bloomにチェック

・エフェクトのPlay再生(光るようになる)

９．ゲームオーバーを表示しよう！
